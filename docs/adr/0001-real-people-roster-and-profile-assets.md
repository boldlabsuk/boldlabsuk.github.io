# Real People Roster and Profile Assets

The Our People page will replace placeholder people with the Website Roster derived from `our_people.json`: main-sheet people with profile pictures, excluding Slack-only additions and rows explicitly marked not to list on the BOLD website. The previous public People Section taxonomy is replaced by Principal Investigator, Postdoc, PhD Student, Masters Student, Associate Members, and Alumni, with Alumni reserved until an explicit alumni source exists. Public profile images should be generated as web-safe public assets from the existing `context` files, including converting unsupported formats such as HEIC or PDF, while leaving the original context files unchanged; spreadsheet Gmail addresses are internal admin data and are not published as public email links.
